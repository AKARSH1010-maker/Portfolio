## Overview

A production-ready task management application featuring complete CRUD operations, intelligent task filtering, and a clean responsive interface. The project demonstrates full stack development skills with a clear separation between frontend and backend.

## Key Features

- **CRUD Operations**: Create, read, update, and delete tasks with a seamless user experience.
- **Task Filtering**: Filter tasks by status, priority, and other criteria for efficient task management.
- **REST API Integration**: Backend API built with Express.js, consumed by the React frontend via Axios.
- **Modular Architecture**: Environment-based configuration and modular services architecture for maintainability.

## Technologies Used

- **React.js & Vite**: For a fast, modern frontend with hot module replacement.
- **Node.js & Express.js**: For a RESTful backend API with structured routing.
- **Axios**: For HTTP client communication between frontend and backend.
- **Railway**: For production deployment of both frontend and backend services.

## Challenges and Learnings

Connecting frontend and backend with proper environment configuration across development and production was a key challenge. Setting up CORS, proxy configuration, and environment variables for deployment on Railway taught practical DevOps skills.

## Outcome

A fully deployed task management app demonstrating end-to-end full stack development, from API design to production deployment.
