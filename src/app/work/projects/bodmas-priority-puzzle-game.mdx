## Overview

An educational game designed to help users practice and master BODMAS (order of operations) through interactive puzzles with increasing difficulty, real-time scoring, and a competitive leaderboard.

## Key Features

- **Dynamic Problem Generation**: Algorithmically generates BODMAS problems with varying difficulty levels.
- **Real-Time Feedback**: Instant visual feedback on correct and incorrect answers to reinforce learning.
- **Scoring System & Leaderboard**: Point-based scoring with a persistent leaderboard for competitive play.
- **Data Persistence**: JSON-based data storage with PHP backend for leaderboard management.

## Technologies Used

- **HTML & CSS**: For the game interface and responsive styling.
- **JavaScript**: For game logic, problem generation, and user interaction.
- **PHP**: For backend processing and leaderboard data management.
- **JSON**: For structured data storage and retrieval.

## Challenges and Learnings

Generating mathematically valid BODMAS problems with predictable difficulty scaling required careful algorithm design. Implementing a backend leaderboard with PHP and JSON storage was a practical exercise in server-side development without a full database.

## Outcome

An engaging educational tool that makes learning BODMAS fun through gamification, with a complete frontend-backend architecture.
