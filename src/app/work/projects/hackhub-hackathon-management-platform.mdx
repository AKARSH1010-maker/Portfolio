## Overview

HackHub is a full stack hackathon management platform designed to streamline event organization for both administrators and participants. It features role-based dashboards, real-time coordination, and a modern SaaS-style interface.

## Key Features

- **Role-Based Dashboards**: Separate Admin and Participant views with tailored functionality for event management and participation.
- **Real-Time Coordination**: Live updates for announcements, team management, leaderboard tracking, and event scheduling powered by Firestore.
- **Secure Authentication**: Google Authentication via Firebase Auth with protected routes and role-based access control.
- **Modular Architecture**: Dynamic routing, reusable component architecture, and responsive design built with Tailwind CSS.

## Technologies Used

- **Next.js & React**: For server-rendered pages and modular component-based UI.
- **Tailwind CSS**: For utility-first responsive styling with a SaaS-inspired design system.
- **Firebase & Firestore**: For authentication, real-time database, and cloud storage.
- **JavaScript**: Core application logic and client-side interactivity.

## Challenges and Learnings

Designing a role-based system that cleanly separates admin and participant experiences while sharing a common codebase was a key challenge. Implementing Firestore security rules to enforce access control at the database level was an important learning experience in building secure full stack applications.

## Outcome

HackHub provides a complete platform for organizing and participating in hackathons, with real-time updates, secure access, and a polished responsive interface ready for production use.
