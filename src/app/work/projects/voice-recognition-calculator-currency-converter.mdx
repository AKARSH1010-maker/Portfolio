## Overview

A feature-rich calculator application with voice recognition capabilities, supporting arithmetic and trigonometric operations, keyboard input, local storage history, and real-time currency conversion via API integration.

## Key Features

- **Voice Recognition**: Multilingual voice input using the Web Speech API for hands-free calculation.
- **Advanced Operations**: Support for arithmetic, trigonometric, and scientific calculations.
- **Keyboard Support**: Full keyboard input support for efficient usage alongside voice and click interactions.
- **Currency Conversion**: Real-time currency conversion powered by REST API integration.
- **Calculation History**: Persistent calculation history using LocalStorage for session continuity.

## Technologies Used

- **React.js**: For component-based UI and state management.
- **Web Speech API**: For voice recognition and multilingual input.
- **JavaScript**: For calculation logic and API integration.
- **REST API & LocalStorage**: For real-time currency data and persistent history.

## Challenges and Learnings

Integrating the Web Speech API across different browsers and handling multilingual voice input required careful error handling and fallback strategies. Managing state between voice input, keyboard input, and button clicks in a unified way was a useful exercise in React state management.

## Outcome

A polished, accessible calculator application that demonstrates creative use of browser APIs and responsive React development.
